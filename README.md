** DoGo – Humanitarian Aid Coordination System**

 📖 Project Overview
DoGo – Humanitarian Aid Coordination System is a web-based application that connects donors and NGOs through a centralized platform. It enables online donation submission, real-time donation tracking, and efficient management of humanitarian aid activities.

❗ Problem Statement
Traditional humanitarian aid systems rely on manual processes, making donation management slow and inefficient. Donors cannot easily track their donations, while NGOs face challenges in maintaining donation records and ensuring transparency.

🎯 Objectives
- Develop a centralized platform for donors and NGOs.
- Enable online donation submission and tracking.
- Improve transparency, accountability, and efficiency in humanitarian aid coordination.

✨ Features
- Secure User Authentication
- Donor Dashboard
- NGO Dashboard
- Online Donation Submission
- Donation Tracking using Tracking ID
- Donation Status Updates
- Donation History
- SQLite Database Integration
- Responsive Web Interface

 ⚙️ Setup / Installation Instructions

 **Clone the Repository**
https://github.com/mjanani18/DoGo-Humanitarian-Aid-Platform.git

**Navigate to the Project Folder**
cd DoGo-Humanitarian-Aid-Platform

** Install Required Packages**
pip install -r requirements.txt

 **Run the Application**
python app.py

** Open in Browser**
```
http://127.0.0.1:5000
```
